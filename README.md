## Installation notes

### To run in docker:

`docker-compose up`

http://localhost:20080/

### To set up:

`init`

`init-roles-and-users`

### To seed demo-data:

`init-demo-data`

### To activate profile after registration without email

```
php yii user/activate {login}
```
